# Security Policy

Please do not report security issues through public GitHub issues or discussions.

## Report A Vulnerability

Send security reports to:

- [developers@liveroom.xyz](mailto:developers@liveroom.xyz?subject=React%20Immersive%20Security%20Report)

Include:

- A short description of the issue
- Steps to reproduce
- Affected package version
- Impact assessment if known
- Any relevant logs, screenshots, or proof-of-concept details

If your report includes customer data or proprietary assets, keep them in the
private email thread only.

## Non-Security Private Support

For license, account, billing, or confidential customer issues, email:

- [developers@liveroom.xyz](mailto:developers@liveroom.xyz?subject=React%20Immersive%20Support)
