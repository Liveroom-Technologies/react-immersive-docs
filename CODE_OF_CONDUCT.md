# Code Of Conduct

This repo is intended to be a respectful, useful public space for customers and
developers using `@live_room/react-immersive`.

## Expected Behavior

- Be respectful and constructive
- Share reproducible details when reporting problems
- Assume good intent and focus on the work
- Keep discussions on-topic and helpful

## Unacceptable Behavior

- Harassment, abuse, or personal attacks
- Sharing private credentials or proprietary customer assets
- Spamming, trolling, or derailing threads
- Posting security vulnerabilities publicly

Maintainers may moderate, edit, lock, or remove content that does not support a
healthy working environment.
