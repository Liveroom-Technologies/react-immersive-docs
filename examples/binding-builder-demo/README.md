# binding-builder-demo

This example mirrors the `BindingBuilder` demo in a public Vite app.

## Run

```bash
npm install
VITE_LICENSE_KEY=your-license-key npm run dev
```

Create a key at [react-immersive.liveroom.dev/console](https://react-immersive.liveroom.dev/console).

It is desktop-first and meant for larger screens.
